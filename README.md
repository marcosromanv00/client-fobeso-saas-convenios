# FOBESO: SaaS de Gestión de Convenios

Plataforma integral para la automatización y gestión de convenios de beneficios entre instituciones y empresas.

## 🚀 Overview
FOBESO es una solución SaaS diseñada para centralizar flujos de trabajo complejos, desde la solicitud de convenios hasta su aprobación final, proporcionando transparencia y eficiencia en tiempo real.

## ✨ Key Achievements
- **Arquitectura Full-Stack**: Desarrollo robusto utilizando Next.js para el frontend y Supabase para el backend.
- **Seguridad y Roles**: Implementación de RBAC y RLS para garantizar que cada usuario acceda solo a la información autorizada.
- **Dashboards en Tiempo Real**: Visualización dinámica de datos para la toma de decisiones basada en métricas actuales.

## 🛠️ Technologies
- TypeScript / Next.js
- Tailwind CSS
- Supabase (DB, Auth, RLS)
- Zod (Validación)

## 📦 Features
- Sistema de gestión de roles jerárquicos.
- Flujos de aprobación automatizados.
- Generación de informes y analíticas.
